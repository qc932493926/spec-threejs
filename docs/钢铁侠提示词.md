### 角色设定
你是资深前端工程师，任务是“复刻并重现现有 JARVIS 全息 HUD 项目”的代码与行为。务必按下述结构、库版本、交互参数与视觉规范生成完整可运行的 Vite + React + TS 项目。

### 项目结构与入口：
- 根入口：`index.html` 使用 Tailwind CDN 与 ImportMap（无需本地 Tailwind 配置）
- 前端入口：`index.tsx` 挂载 `App` 到 `#root`
- 主界面：`App.tsx` 包含“启动页 → Intro → 主界面”三段式流程
- 组件：`components/VideoFeed.tsx`、`components/HolographicEarth.tsx`、`components/HUDOverlay.tsx`、`components/JarvisIntro.tsx`
- 服务：`services/mediapipeService.ts`（手势识别初始化），`services/soundService.ts`（Web Audio 程序化音效 + TTS）
- 类型：`types.ts`（Landmark、HandInteractionData、RegionName 等）
- 模型资源：`modules/gesture_recognizer.task`（MediaPipe 手势模型，支持本地/远程双路径）
- 构建：`vite.config.ts`（别名 `@` 指向项目根，`assetsInclude` 包含 `.task`）

- 顶部左侧：系统状态牌与随机十六进制滚动日志

### 3D 全息地球（R3F + Three + PostProcessing）：
- 地球材质：`meshPhongMaterial`，`map + normalMap + specularMap`，`AdditiveBlending` 叠加发光（发光映射陆地）
- 包装：云层球（轻透明）、反向旋转线框（icosahedron wireframe）、外环（ring）、环绕卫星粒子（`Points` + `PointMaterial`）
- 光效：Bloom（`@react-three/postprocessing`）营造全息感
- 旋转面向区域判定并回传中文地区名（美洲/太平洋/亚洲/欧洲/非洲）
- 左手展开度控制各层半径与自转速度差异，右手控制地球 yaw/pitch

### 手势与交互（MediaPipe 双手）：
- 识别：`GestureRecognizer`，`numHands = 2`，`runningMode = VIDEO`，首选本地模型，失败走远程 CDN；WASM 优先 jsDelivr，失败切 unpkg
- 左手展开度（expansionFactor）：按拇指尖(4)与食指尖(8)距离归一化；`minPinch=0.02 / maxPinch=0.18` 映射 0~1
- 右手旋转控制：取中指 MCP(9)位置映射 `x∈[-1,1]` 控水平旋转、`y∈[-1,1]` 控俯仰倾斜
- 右手捏合：拇指尖(4)与食指尖(8)距离 `<0.05` 判定 pinch，显示悬浮情报面板并绘制连接线，松开隐藏
- 左手腕附近绘制圆形“展开度/引擎出力”仪表盘，超过 95% 显示红色“最大输出”
- 指尖绘制旋转锁定圈；骨架用虚线；掌心显示手 ID 文案

### 地形切换与战术视图：
- 左手展开度平滑过渡（指数滤波），`exp > 0.4` 起对地球淡出，`exp > 0.55` 触发“地形模式”音效并显示战术地形
- 战术地形：高细分 `PlaneGeometry` 用简易 FBM/正弦组合生成起伏；线框+半透明填充；同心雷达环随时间衰变透明；波峰处悬浮目标标记（圈、十字、竖线）
- 右手旋转同样影响战术地形整体旋转/倾斜，以保持技术感

### 程序化音效（Web Audio + TTS）：
- 点击/细微交互：`playBlip()`
- 机械锁定/释放：`playLock()` / `playRelease()` 对应 pinch 显示/隐藏
- 展开度移动：`playServo(intensity)` 按帧变化触发短粒度电机感
- 地形切换：`playMapSwitch()`
- 环境：`playAmbientHum()` 持续低频嗡鸣
- TTS：`speak("Hello. I am Jarvis.")`，偏英式男声；在引导完成后播放

### 性能与渲染策略：
- 摄像头与识别：`requestAnimationFrame` 驱动，`video.currentTime` 比较防止重复推理；手动取消/释放摄像头轨道
- 3D：`useFrame` 更新旋转/缩放/淡入淡出，粒子与环轻量动画；`gl.antialias=false` 以配合后期
- UI Canvas：自适应窗口大小，合并绘制，减少状态更新
- 降级：摄像头或音频失败时保留 UI 与动画，给出日志提示
