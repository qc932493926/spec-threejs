# 常用开发命令

## 开发运行
```bash
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run preview      # 预览生产版本
```

## 代码质量
```bash
npm run lint         # 运行ESLint检查
```

## 测试
```bash
npm run test         # 运行测试(监视模式)
npm run test:run     # 运行测试(单次)
npm run test:ui      # 测试UI界面
npm run test:coverage # 测试覆盖率
```

## Git提交规范
```bash
git add .
git commit -m "vXX: 功能描述"
```

## Windows系统命令
- 使用 `dir` 代替 `ls`
- 使用 `type` 代替 `cat`
- 路径使用反斜杠 `\` 或正斜杠 `/`
