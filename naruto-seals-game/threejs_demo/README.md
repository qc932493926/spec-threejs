# Three.js Demo

一个基于 Three.js 的 3D 图形演示项目。

## 项目简介

这是一个简单的 Three.js 示例项目，展示了一个旋转的 3D 立方体。

## 功能特性

- ✅ 3D 立方体渲染
- ✅ 自动旋转动画
- ✅ 响应式设计
- ✅ 使用 Vite 构建工具

## 技术栈

- **Three.js** - 3D 图形库
- **Vite** - 快速开发构建工具
- **JavaScript ES6+** - 现代 JavaScript 语法

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

然后访问 `http://localhost:5173`

### 生产构建

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
